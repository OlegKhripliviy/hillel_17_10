## Quick start

### Install deps

```bash
# Install pipenv
pip install pipenv

# Activate virtual env
pipenv shell

#Install deps
pipenv sync
```

#### Additional
```bash
# Regenerate Pipfile.lock file
pipenv lock

# pipenv lock & pipenv sync
pipenv update
```
